# MuseScore 4 → 3 変換ツール

MuseScore 4で作成された楽譜データ（`.mscz` ファイル）を、旧バージョンの MuseScore 3 で読み込める形式にダウングレード変換する、ブラウザ完結型のWebアプリケーションです。

## 主な機能

* **バージョンのダウングレード**: MuseScore 4専用のファイル構造と内部バージョン情報を、MuseScore 3（v3.6.2互換）のものに書き換えます。
* **タグの互換性修正**: MuseScoreのメジャーアップデートに伴って変更された内部XMLタグ（`<concertKey>` や `<style>` など）を、旧バージョンで認識できる形式に置換します。
* **ブラウザ完結**: すべての処理はローカルのブラウザ内で完結し、外部サーバーへのデータ送信は行いません。機密性の高い未発表の楽譜などでも安全に変換可能です。

## 注意事項

* **レイアウトや新機能の欠落**: MuseScore 4で新しく追加された機能（新しいレイアウトエンジンによる配置、VTSプラグインの設定、MuseScore 4専用のテキスト・記号など）は、MuseScore 3には存在しないため、変換後にレイアウトが崩れたり、一部のデータが欠落・変形したりする可能性があります。変換後は必ずMuseScore 3上でレイアウトの微調整を行ってください。
* **上書きの防止**: 変換後のファイルは、元のファイル名に `_msc3` という接尾辞が付与されてダウンロードされるため、元のデータを誤って上書きすることはありません。

## 内部の処理フロー

`.mscz` ファイルは、実態としてはXMLファイル（楽譜データ本体）などを圧縮したZIPファイルです。本ツールでは `JSZip` ライブラリを使用し、以下の手順でファイルを直接書き換えています。

1. **ファイルの解凍とパス特定**
   アップロードされた `.mscz` (ZIP形式) をメモリ上で解凍し、`META-INF/container.xml` を参照して楽譜データ本体（`.mscx` ファイル）のパスを特定します。

2. **バージョンとタグの書き換え**
   抽出した `.mscx` ファイル（XML文字列）に対して、正規表現を用いた置換処理を行い、MuseScore 3仕様に適合させます。
   * `<style>` タグ内の要素名の先頭を大文字化
   * `<concertKey>` タグを `<accidental>` タグに変換
   * `museScore version` を `"3.02"` に変更
   * `<programVersion>` を `3.6.2` に、`<programRevision>` を `3224f34` に偽装

3. **再圧縮と出力**
   書き換えた `.mscx` データと、新しく生成した `container.xml` のみを格納した新しいZIPアーカイブを作成し、`.mscz` ファイルとして出力・ダウンロードさせます。

## 使用ライブラリ

* [JSZip](https://stuk.github.io/jszip/) (via cdnjs) - ZIPファイルの読み込み、作成、圧縮処理
